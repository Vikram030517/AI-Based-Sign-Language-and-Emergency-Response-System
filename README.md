AI-Based Sign Language and Emergency Response System
Duration: March 2026 – June 2026
Overview
Developed a real-time AI-based system to recognize predefined sign language gestures using OpenCV. The system also detects emergency gestures and triggers alert-based responses. Additionally, it includes face analysis features for age, gender, and emotion recognition.
Technologies Used:
Python
OpenCV
Machine Learning
NumPy
Key Features:
Real-time sign language gesture recognition.
Emergency gesture detection with instant alerts.
Face detection for age, gender, and emotion prediction.
Live webcam-based processing.
Outcome:
Improved communication accessibility and provided a faster emergency response mechanism through computer vision.
